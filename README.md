# Portfolio

Personal portfolio site built with **React**, **TypeScript**, **Vite**, **pnpm**, **Ant Design**, and **React Router**.

## Scripts

```bash
pnpm install
pnpm dev
pnpm build
pnpm preview
```

## Structure

- `src/pages/` — route screens (Home, About, Projects, Contact)
- `src/components/MainLayout.tsx` — shell with navigation and footer
